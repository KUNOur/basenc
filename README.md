Basenc
======

Basenc is a library used to encode data into a human-readable encoding format.

Supported Encodings
-------------------

- Base32

How to Build
------------

This project uses the CMake build system. To build the basenc library
(libbasenc), do the following in the top-level directory:

	mkdir build
	cd build
	cmake ..
	make
